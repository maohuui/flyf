# flyf
简单FTP客户端，取名含义fly+ftp，飞快的ftp

# 已支持的命令
* ls     显示服务端文件列表
* lls    显示本地文件列表
* pwd    显示服务端工作目录
* lpwd   显示本地工作目录
* cd     更改服务端工作目录
* lcd    更改本地工作目录
* get    下载文件
* put    上传文件
* delete 删除服务端文件
* ascii  更改传输模式为ASCII
* binary 更改传输模式为BINARY
* mkdir  服务端新建文件夹
* help   帮助信息
* open   连接服务端
* exit   退出程序

# 编译
make
